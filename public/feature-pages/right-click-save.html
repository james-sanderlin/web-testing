Right-Click Save Image

Reproduces the "Right-click → Save Image" flow from the Notary Journal report. The image below is embedded as a data:image/jpeg;base64,… src — no HTTP request is made to fetch it. Right-click the image and choose Save Image As… to a Secure Storage destination, then try to open it in Secure Viewer.

Steps:
  1. Wait for the image to load below (status will update)
  2. Right-click the image
  3. Choose Save Image As… and save to a Secure Storage folder
  4. Open the saved file in Secure Viewer
